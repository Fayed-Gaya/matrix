# Balls

## Overview

Balls is a simple movement experiment. It draws two red circles, moves them with
keyboard input, and wraps them around the screen edges.

## Run

From the repository root:

```bash
python3 -m levels.balls
```

## Controls

- `W`: move up.
- `S`: move down.
- `A`: move left.
- `D`: move right.
- Close the window to quit.

## Files

- `balls.py`: standalone Pygame loop, movement logic, and wraparound drawing.

## Notes

This is a demo/experiment rather than a complete game.
