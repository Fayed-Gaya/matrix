# Bouncing Ball

## Overview

Bouncing Ball is a minimal Pygame demo. It loads a ball image and moves it
around the window, bouncing when it reaches the screen edges.

## Run

This level currently loads `intro_ball.gif` from the current working directory,
so run it from the level directory:

```bash
cd levels/bouncing_ball
python bouncing_ball.py
```

## Controls

- Close the window to quit.

## Files

- `bouncing_ball.py`: standalone Pygame loop and bounce logic.
- `intro_ball.gif`: ball image asset.

## Notes

This is a minimal demo rather than a complete game.
