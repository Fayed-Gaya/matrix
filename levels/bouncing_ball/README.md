# Bouncing Ball

## Overview

Bouncing Ball is a minimal Pygame demo. It loads a ball image and moves it
around the window, bouncing when it reaches the screen edges.

## Run

From the repository root:

```bash
python3 -m levels.bouncing_ball
```

## Controls

- Close the window to quit.

## Files

- `bouncing_ball.py`: standalone Pygame loop and bounce logic.
- `intro_ball.gif`: ball image asset.

## Notes

This is a minimal demo rather than a complete game. It uses paths based on the
location of `bouncing_ball.py`, so it can run from the repository root.
